# 🎨 Catálogo André

Plataforma web para exibição e venda de quadros decorativos.

## 🚀 Funcionalidades
- Exibição de produtos com imagem, tamanho e preço
- Seleção de variações (tamanho)
- Carrinho simples
- Finalização de pedido via WhatsApp

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## 📱 Objetivo
Facilitar a venda de quadros de forma simples e direta, sem necessidade de e-commerce complexo.
